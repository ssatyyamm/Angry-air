⚠️ TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Music doesn't play?
   ✔️ Make sure file is named exactly: perfect.mp3
   ✔️ Click anywhere on the page (browser autoplay rules)
   ✔️ Check file format is MP3, not M4A or other

❌ Photos don't show?
   ✔️ File names must match exactly (case-sensitive!)
   ✔️ Use JPG or PNG format
   ✔️ Keep files in the same folder as HTML

❌ Page looks broken?
   ✔️ Make sure all 3 files are together:
      - romantic-apology-enhanced.html
      - styles.css
      - script.js
   ✔️ Don't rename these core files!

❌ Buttons don't work?
   ✔️ Make sure script.js is in the same folder
   ✔️ Try a different browser
   ✔️ Check browser console (F12) for errors
